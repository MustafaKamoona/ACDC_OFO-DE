# Hybrid AC/DC Microgrid: PID vs OFO vs OFO-NN (Option B)

Hybrid AC/DC microgrid simulation with PV + Wind, one DC load, two VSCs, two AC loads, and bidirectional power exchange with the utility grid.

Constraints
- DC bus: Vdc = 850 ± 8 V (842..858 V)
- AC buses: V_phase_rms = 230 ± 10 V (220..240 Vrms)

Controllers
- PID reference (auto-tuned)
- OFO (slow re-learning of the PID set-up)
- OFO-NN (online disturbance estimator + OFO, fast convergence)

Option B
- Averaged dq power-dynamics model for 24h-compressed simulation
- Switching EMT window for THD (short time segment)

Quickstart
1. Install requirements:
   pip install -r requirements.txt
2. Auto-tune PID:
   python -m hybrid_acdc autotune --config config/system.yaml --out results/pid_ref --max_evals 48
3. Run OFO and OFO-NN iterations:
   python -m hybrid_acdc iterate --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo --out results/ofo
   python -m hybrid_acdc iterate --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo_nn --out results/ofo_nn
4. THD window (EMT):
   python -m hybrid_acdc thd --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo_nn --out results/thd
