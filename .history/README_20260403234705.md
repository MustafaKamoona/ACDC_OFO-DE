# Hybrid AC/DC Microgrid: PID vs OFO vs OFO-DE
# Model-Less Online Feedback Optimization with Disturbance Estimation for AC/DC Microgrids

Hybrid AC/DC microgrid simulation with PV + Wind, one DC load, two VSCs, two AC loads, and bidirectional power exchange with the utility grid.

Constraints
- DC bus: Vdc = 850 ± 8 V (842..858 V)
- AC buses: V_phase_rms = 230 ± 10 V (220..240 Vrms)

Controllers
- PID reference (auto-tuned, well-tuned)
- OFO (slow re-learning of the PID set-up)
- OFO-DE (online disturbance estimator + OFO, fast convergence)

- Averaged dq power-dynamics model for 24h-compressed simulation
- Switching EMT window for THD (short time segment)

## 🔍 Overview
The proposed control approach enables:
- DC-link voltage regulation without explicit system modeling  
- Adaptive power sharing between multiple converters  
- Real-time compensation of power imbalances using online disturbance estimation  

The framework combines:
- Online Feedback Optimization (OFO)  
- Lightweight online disturbance estimation  
- PI-based outer-loop control  

## ⚡ Key Features
- Fully model-free control design  
- Fast convergence compared to conventional OFO  
- Robust performance under time-varying renewable generation and load conditions  
- Suitable for real-time microgrid control  

## 🧪 Simulation Setup
- 24-hour time-domain simulation  
- Hybrid AC/DC microgrid with:
  - PV and wind generation  
  - AC and DC loads  
  - Two bidirectional VSCs  

## 📊 Results
The proposed OFO-DE controller:
- Achieves faster convergence (~50% improvement)  
- Maintains stable DC-link voltage regulation  
- Provides effective power sharing and grid interaction

Quickstart
1. Install requirements:
   pip install -r requirements.txt
2. Auto-tune PID:
   python -m hybrid_acdc autotune --config config/system.yaml --out results/pid_ref --max_evals 48
3. Run OFO and OFO-NN iterations:
   python -m hybrid_acdc iterate --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo --out results/ofo
   python -m hybrid_acdc iterate --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo_nn --out results/ofo_nn
4. THD window (EMT):
   python -m hybrid_acdc thd --config config/system.yaml --pid_ref results/pid_ref/pid_ref.json --controller ofo_nn --out results/thd
